# Strategy-game
Programming Technologies course project

### There is fractions with hierarchy:
### Elf
	single unit

### Healer
	elf with heal

### Warrior
	healer + more stats

### Mage
	elf with magic

### Archmage
	healer + mage

## Installation
Download from repo
```
	git clone https://github.com/SolomidHero/Strategy-game
```
Make executable file
```	
	mkdir build
	cd build
	cmake ..
	make
```
Run
```
	cd build
	./game
```
